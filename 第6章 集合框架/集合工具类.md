Arrays.copyOf()

System.arraycopy()

```java
public static native void arraycopy(Object src,  int  srcPos,  
                                        Object dest, int destPos,  
                                        int length);  
```

- src - 源数组。 
- srcPos - 源数组中的起始位置。 
- dest - 目标数组。 
- destPos - 目标数据中的起始位置。 
- length - 要复制的数组元素的数量。
